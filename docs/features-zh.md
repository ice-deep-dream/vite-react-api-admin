## 📝 核心功能明细

### 1. 项目管理

#### 功能描述
支持配置多个后端项目的 Swagger/OpenAPI 地址，实现快速切换和管理。

#### 功能点

| 功能 | 描述 |
|------|------|
| 添加项目 | 输入项目名称和 Swagger JSON 地址，自动解析并加载 API 数据 |
| 编辑项目 | 修改已有项目的名称或 Swagger 地址 |
| 删除项目 | 移除不再需要的项目配置 |
| 切换项目 | 在不同项目间快速切换，自动加载对应 API 数据 |
| 测试连接 | 保存前测试 Swagger 地址是否可访问 |

#### 操作流程

```
用户点击设置图标
    ↓
打开设置面板
    ↓
点击 "+" 添加项目
    ↓
输入项目名称和 Swagger 地址
    ↓
点击 "保存并测试"
    ↓
系统验证地址 → 成功 → 解析 Swagger 数据 → 加载 API 列表
              ↓
            失败 → 显示错误信息
```

### 3. API 调试

#### 功能描述
支持发送真实 HTTP 请求，实时查看响应结果。

#### 功能点

| 功能 | 描述 |
|------|------|
| 方法选择 | 支持 GET/POST/PUT/DELETE/PATCH |
| URL 编辑 | 可修改请求 URL |
| 参数编辑 | Query/Path 参数键值对输入 |
| Headers 管理 | 自定义请求头，支持全局 Token 自动注入 |
| Body 编辑 | JSON 格式请求体编辑，支持语法高亮 |
| 响应展示 | 状态码、响应时间、响应体（JSON 格式化） |
| Token 自动保存 | 登录接口成功后自动提取并保存 Token |

#### 调试流程

```
选择 API 接口
    ↓
填写请求参数
    ↓
编辑 Headers（可选）
    ↓
编辑 Body（POST/PUT/PATCH）
    ↓
点击 "发送" 按钮
    ↓
发送 HTTP 请求 → 等待响应
    ↓
展示响应结果
    ↓
如果是登录接口且成功 → 自动保存 Token
```

### 4. 认证管理

#### 功能描述
内置登录功能，支持 Bearer Token 认证，自动保存和管理 Token。

#### 功能点

| 功能 | 描述 |
|------|------|
| 登录表单 | 用户名、密码、验证码输入 |
| 验证码 | 自动获取图片验证码，支持点击刷新 |
| Token 提取 | 自动从登录响应中提取 Token |
| Token 存储 | 使用 Zustand 持久化存储 |
| 自动注入 | 后续请求自动添加 Authorization Header |
| 退出登录 | 清除 Token 和认证状态 |

#### 认证流程

```
打开登录面板
    ↓
输入用户名、密码、验证码
    ↓
点击 "登录"
    ↓
发送 POST /api/auth/login
    ↓
验证响应 → 成功 → 提取 Token → 保存到全局状态
          ↓                    ↓
        失败 → 显示错误信息    后续请求自动携带 Token
```
